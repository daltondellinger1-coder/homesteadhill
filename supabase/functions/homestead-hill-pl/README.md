# Homestead Hill P&L live-data endpoint

This edge function serves the admin-only `/admin/pl` dashboard.

## Security

- Requires a signed-in Supabase user session.
- Confirms the user is in `admin_allowlist` using the service role key server-side.
- Never exposes QuickBooks, Composio, or service-role secrets to the browser.

## Current behavior

By default, it returns `snapshot.json`, which matches the React dashboard data shape.

## Live refresh hook

To make it pull a refreshed JSON artifact generated by the weekly Hermes/QBO refresh job, set Supabase secrets:

```bash
supabase secrets set HOMESTEAD_HILL_PL_LIVE_JSON_URL="https://.../homestead-hill-pl.json"
supabase secrets set HOMESTEAD_HILL_PL_LIVE_JSON_TOKEN="optional-bearer-token"
```

The JSON at that URL must match `src/data/homesteadHillPlSnapshot.ts` shape:

- `dateRange`
- `baselineMonthlyExpenses`
- `summary`
- `months`
- `transactions`
- `unitMonth`
- `review`

If the live URL is missing, unavailable, or returns the wrong shape, the function falls back to the embedded snapshot and logs the failure server-side.
